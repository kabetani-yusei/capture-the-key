# Capture The Key

## 文章を要約するアプリ
文章を入力して「要約する」ボタンを押すと、
API にリクエストを送り、要約した文章を返すアプリです。

## 日記

### 2026/04/22
最近話題の Claude Code で、文章を要約するアプリを作ってみる。  
作ってほしいアプリの概要を入力したら、Claude Code が全部作ってくれた。すごい！  
最初はエラーが起きて、やっぱり一発では無理かと思っていたけど、  
[src/app/page.tsx](src/app/page.tsx) の `KEY2 = 'ここにあなたのKEY2を入れる'` をそのまま実行していたのが原因だった。  
自分の KEY2 を入れたらちゃんと動いて、動くものが作れて感動した！

### 2026/04/23
どうやらシークレットキーなどの外部に公開されたら困るものを、直接コードに書き込むのは危険らしい。  
そこで、これまで使っていた KEY2 は使えないようにして、新しく KEY3 を発行することにした。  
[.env](.env) ファイルを新しく作り、キーはそちらに書いて、[src/app/page.tsx](src/app/page.tsx) からは `process.env` 経由で読み込むようにしてみた。  
これでコードに直接キーを書かなくて済むので安心！
